## ğŸ”§ Cara Menjalankan Proyek

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Jalankan aplikasi**

   ```bash
   npx expo start
   ```

   Setelah itu, kamu bisa memilih untuk membuka aplikasi di:
